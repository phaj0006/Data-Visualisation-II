# Data-Visualisation-II
A 11 chart presentation on Tourism in Australia

This is made by Peter Hajduk on 31/05/2026
To run this project, simply run the index.html file whilst keeping all the other files in the same structure.

This visualisation uses the following datasets for the information:
https://my.ibisworld.com/au/en/industry/X0003/about
https://my.ibisworld.com/au/en/industry/H4401/about
https://my.ibisworld.com/au/en/industry-specialized/OD5397/about

For the map, Rowanhogan’s Australia States geojson map was used, which was also converted to a topojson file
This is found here: https://github.com/rowanhogan/australian-states/blob/master/states.geojson

The frutiger font comes from: https://font.download/font/frutiger
